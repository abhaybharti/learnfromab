# HashMapPrint Values

## Code to print HashMap key/value in console

```
import java.util.*;
public static void main(String[] args) {
         Map<Integer, String> map1 = new HashMap<Integer, String>();
               map1.put(1, "david");
                map1.put(2, "simon");
                map1.put(3, "mary");
                map1.put(4, "john");
                map1.put(5, "jane");

        System.out.println(map1);
    }
```

`Output : ` {1=david, 2=simon, 3=mary, 4=john, 5=jane}
