# JMeter-Java-DSL

### JMeter DSL to start using JMeterDSL

add following in POM.XML file

```
<dependency>
    <groupId>us.abstracta.jmeter</groupId>
    <artifactId>jmeter-java-dsl</artifactId>
    <version>0.62</version>
    <scope>test</scope>
 </dependency>
```
